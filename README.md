# 📚 StudyHub

A simple and responsive educational website built using **HTML**. StudyHub is designed to provide students with an easy-to-use platform for accessing study-related content and resources.

## 🚀 Features

- 📖 Simple and clean user interface
- 📱 Responsive layout
- 🎓 Student-friendly design
- ⚡ Fast and lightweight
- 🌐 Built with pure HTML

## 🛠️ Technologies Used

- HTML5

## 📂 Project Structure

```
StudyHub/
│
├── index.html
├── README.md
└── assets/
```

## ▶️ Getting Started

1. Clone the repository:

```bash
git clone https://github.com/suryajaiswal047/StudyHub.git
```

2. Navigate to the project folder:

```bash
cd StudyHub
```

3. Open `index.html` in your web browser.

No installation or additional setup is required.
